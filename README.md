# Imatic Sentry

## Purpose

Sends errors to Sentry.

## Installation

```
composer config --unset platform.php
composer require imatic-it/imatic-sentry
```

## Configuration

Env variables that can be used to configure the plugin:
```
IMATIC_SENTRY_DSN
IMATIC_SENTRY_RELEASE
```
